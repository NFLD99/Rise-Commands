## About The Command
This command helps with Updating rise on Linux

## Installation

1. Download the .py file and place it in your Rise scripts folder
1. Download the .sh file and place it in your /home folder
2. Restart Rise

<!-- USAGE EXAMPLES -->
## Usage
Command:
```
[prefix]updaterise
```

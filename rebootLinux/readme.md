## About The Command
This command makes restarting your Linux VPS a little easier

## Installation

1. Download the .py file and place it in your Rise scripts folder
2. Restart Rise

<!-- USAGE EXAMPLES -->
## Usage
Command:
```
[prefix]rebootlinux
```
